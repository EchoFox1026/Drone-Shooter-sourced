# Drone Shooter [sourced]
### Source code for the game "Drone Shooter" published by Starfrost Media.
This game was recently discontinued by Starfrost Media and will be the first to have the source code published publicly. We know web games can be lost to time if the servers hosting them go down, and in a reaction to that very thing happening to our servers, we will now be publishing all discontinued game source code here on our GitHub page alongside compiled desktop versions made with Tauri/Electron (not decided yet).
